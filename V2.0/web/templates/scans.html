Each scan point will capture 2 images at these focus positions
📖 Focus Mode Descriptions
  • Autofocus Initial: Trigger autofocus once at the start of the scan, then use that focus for all points (calibration/setup)
  • Continuous Autofocus: Enable continuous autofocus for all scan points (slower but adapts to varying distances)
  • Manual Focus: Set a fixed lens position for all scan points (fastest, consistent focus)
  • Manual Focus Stacking: Capture multiple images at different focus positions for each scan point (depth-of-field stacking)
